# Lähdekoodin hankkiminen

Jos haluat tehdä muutoksia Servoon tai kääntää servoshellin itse, kloonaa [päärepo](https://github.com/servo/servo) Gitillä:

```sh
$ git clone https://github.com/servo/servo.git
$ cd servo
```

<div class="warning _note">

**Servon repo on melko suuri!**
Jos verkkoyhteys on epävakaa tai levytila rajallinen, harkitse [matalaa kloonausta](https://github.blog/2020-12-21-get-up-to-speed-with-partial-clone-and-shallow-clone/).

</div>


Kun olet kloonannut Servon lähdekoodin, [seuraava vaihe on riippuvuuksien asentaminen ja Servon kääntäminen](building.md).
